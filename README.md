# VereinsKalender

Schedule management system for organization groups.

## Quick Start

```bash
npm install
npm run web:dev      # Start web app
npm run mobile:dev   # Start mobile app (Expo)
```

## Features

- 📱 Cross-platform: Web (React PSA) and Mobile (React Native + Expo)
- 📅 Complex scheduling with recurring patterns
- 🗺️ Regional holiday management
- 📤 Export to iCalendar and Excel formats
- 🎯 Group-based filtering and relevance
- 💾 Local storage with optional Nextcloud integration
- 📋 Demo calendar included by default

## Project Structure

- `packages/web` - Web application (React PSA)
- `packages/mobile` - Mobile application (React Native + Expo)
- `packages/shared` - Shared utilities and types
